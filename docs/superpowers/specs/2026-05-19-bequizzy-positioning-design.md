# BeQuizzy Positioning Design

## Summary
BeQuizzy should start as an AI-native career gym and hiring evidence platform for **Sales / Business Development** talent, not as a generic non-technical testing suite.

## Product thesis
The product combines:
- continuous candidate practice
- modular multimodal assessments
- structured scorecards
- verified evidence profiles
- employer-side shortlist and comparison workflows

## Recommended wedge
- **Role family:** Sales / Business Development
- **Candidate segment:** junior, early-career, graduate, and career-switching talent
- **First payer:** SMEs hiring sales/BD internally
- **Secondary payer later:** candidates via premium coaching and profile upgrades

## Core loop
mock interview / roleplay → scorecard → diagnosis → next drill → visible progress → stronger evidence profile

## Signature assessment stack
- AI roleplay call
- situational judgment test
- presentation / pitch test
- communication scoring layer

## Key differentiation
BeQuizzy should not position as a one-off assessment tool. It should position as a system that converts repeated practice into employer-trusted evidence.

## Strategic guardrails
- start with one job family
- avoid generic psychometric framing
- prioritize explainable scoring
- expand by role playbooks, not by broad role coverage
